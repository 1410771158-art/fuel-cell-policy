关于本项目

我们的使命与愿景

燃料电池政策速递是一个致力于为燃料电池行业从业者提供及时、全面政策信息的公益性项目。我们每周对国家及各地出台的燃料电池相关政策进行系统性梳理和汇编，涵盖国家政策、地方政策、行业标准、国际动态、财政补贴及技术标准等多个维度。

本网站完全免费开放，所有文档均支持在线阅读和免费下载。我们希望通过这一平台，帮助行业从业者高效获取政策信息，把握行业发展方向。

使用指南

如何获取和部署这个网站

在线阅读

点击文档卡片上的「在线阅读」按钮，即可在弹窗中查看该周政策信息的详细摘要和关键要点。

免费下载

点击「下载文件」按钮即可下载对应文档。将您的PDF文件放到 `docs/` 文件夹，并更新下方数据中的 fileUrl 即可。

免费部署

将代码推送到 GitHub 仓库，启用 GitHub Pages 即可免费部署。也可使用 Netlify 或 Vercel 一键部署，全程零成本。

内容维护

每周只需编辑 HTML 中的 documentsData 数组，添加新文档信息，推送到仓库即可自动更新网站内容。

自定义域名

GitHub Pages 支持绑定自定义域名，只需在仓库设置中配置 DNS 解析，即可使用您自己的域名访问网站。

搜索与筛选

使用页面顶部的搜索栏和分类标签，可快速定位感兴趣的政策文档，支持按关键词和类别筛选。
